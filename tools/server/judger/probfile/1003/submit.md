2. 建立一个**Private**仓库
3. 对OJ测评账户开放访问权限，操作：
    1. Settings->User and group access.
    2. 输入User: Hzfengsy.
    3. 添加 Read 权限.
4. push git仓库至BitBucket.
5. 将repo的**SSH**地址粘贴到提交框.  
特别提示：一定要是**SSH**地址，不能**HTTPS**地址.例：
```

```
